# 疲労シャカシャカ — Design System

縦画面の**疲労可視化アプリ**のためのデザインシステム。「やったこと」を縦に並べるタイムラインと、行動が3D絵文字として背景に積み上がる「シャカシャカ」レイヤーで、自分の心身の疲労を *ちょうどよく* 可視化する。あすけんの「食べ物・カロリー」を「行動・疲労度」に置き換えたイメージ。

> 配色・タイポの“雰囲気”はモバイルゲーム風の搭乗券／チケット表現を参考にした**オリジナル体系**です。特定作品のロゴ・キャラクター・ブランド要素は一切複製していません。

## プロダクトの考え方
- 疲労度は **0〜100**。100で背景が満杯＝要休息。
- 行動で疲労が**加算**、食事・休息・睡眠で**減算**（食事は1時間後などに効果が切れて項目が消せる／睡眠後はまとめて消せる／前日の疲労は持ち越し）。
- ゴールは「頑張りすぎな人がちょうどよく頑張れる」こと。常に **余白（残りキャパ）%** を見せる。

## CONTENT FUNDAMENTALS（コピー）
- 一人称は出さず、ユーザーに**やさしく寄り添う**短文（「まだいける」「そろそろ休もう」「詰め込みすぎ注意」）。
- 英字の見出し・ラベルは**モノスペース大文字**（WORD / TODAY / FATIGUE / REST UP!）。日本語本文はゴシック。
- 数字を主役に。疲労度・時刻は大きく。説教くさくしない。絵文字は“行動”と“気持ち”の表現としてのみ使う。

## VISUAL FOUNDATIONS
- **色**: ライム `#C4F000`（アクション）＋ピンク `#FF5FA2`（強調・今日）。地は暖色寄りクリーム `#F7F4EC`、面は白、文字は近黒 `#1B1B18`。彩度は低めに抑える。疲労ランプは lime→amber→pink の控えめな段階。
- **タイポ**: Zen Kaku Gothic New（和文UI）／ Space Mono（英字キャプション・バーコード風）／ Archivo（大きな英数字）。
- **角丸**: 控えめ（6–18px）。チケットはほぼ角ばり＋小さな切り込み(notch)。
- **影**: 低く柔らかい単方向。3D絵文字には接地影（drop-shadow）。グロー無し。
- **モチーフ**: 黒タブ＋英字ラベル、バーコード、スタンプ風（回転した枠）、切り込み付きチケットカード。
- **背景**: シャカシャカ＝行動絵文字が床から積み上がるレイヤー。疲労度に比例して数が増える。
- **アニメ**: 控えめ。fill幅のトランジション、押下のscale。派手なバウンス無し。

## ICONOGRAPHY
- アイコンは基本 **Unicode絵文字**（💻🚃📅🍙✏️💬🏢／😆🙂😌🥲😩😢）。OS側の3D風レンダリングを活かす。独自SVGは描かない。
- 必要に応じ CDN のラインアイコン（Lucide 等）を同じストローク感で補完してよい。

## ファイル索引
- `styles.css` — 消費側が読む唯一のエントリ（`@import` のみ）。
- `tokens/` — `fonts.css` `colors.css` `typography.css` `spacing.css` `effects.css`
- `guidelines/` — 配色・タイポ・余白・モチーフの specimen カード（Design Systemタブに表示）
- `components/` — `core/`(Button, TicketTag) `fatigue/`(FatigueGauge, MoodFace, ShakaBackground) `timeline/`(TimelineItem)
- `ui_kits/fatigue-app/index.html` — **メイン画面 3案**（Timeline主役 / Shaka主役 / チケット＋疲労チューブ）
- `SKILL.md` — Agent Skill 互換マニフェスト
