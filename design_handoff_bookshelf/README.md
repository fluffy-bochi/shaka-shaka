# 本棚タイムライン — 本番実装ハンドオフ

「がんばりの本棚」（1日=1冊の透明な本に絵文字がたまるタイムライン）を本番 `native/index.html` に実装するための一式。

## ファイル
- `指示書.md` — 仕様書。画面構成・挙動・データマッピング・デザイントークン・絶対原則。**実装前に必ず全部読む**
- `CLAUDE_CODE_PROMPT_bookshelf.md` — Claude Code にそのまま渡す実装プロンプト
- `reference/本棚タイムライン 案.dc.html` — デザインの正（動くモック）。採用は **1a（縦・本棚ビュー）** と **3a（横・採用版）** のみ。TURN 2 の 2a/2b・1b は検討過程なので実装しない
- `reference/support.js` — モックを開くのに必要なランタイム（同じフォルダに置いて 1a/3a を実機確認できる）

## 採用デザインの要点
- 1a = 縦持ち本棚（週=1段・月曜始まり・週番号ゴールド・下からポップアップするその日シート）
- 3a = 横持ち一列棚（疲労と回復⇄寝るまえの量、右に本のページ、月別は表紙にためる、回転で縦横切替）
- 数値・ゲージ・%・警告色は出さない（かめペース絶対原則）
