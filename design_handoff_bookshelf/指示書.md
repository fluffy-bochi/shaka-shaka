# 本棚タイムライン 仕様書（1a 縦・3a 横）

デザインの正: `reference/本棚タイムライン 案.dc.html` の #1a と #3a。迷ったらモックの見た目・挙動に合わせる。

## 0. コンセプト
- **1日 = 1冊の透明な本**。背表紙の中に、その日の記録絵文字が**下（朝）→ 上（夜）**へ積もる
- 絵文字1個 = 疲労1。**過去=実体（カラー）／予定・未消化=グレー（opacity .3 + grayscale .85）**
- 今日 = ライム枠（#c4f000）＋「今日」バッジ。選択中 = ピンク枠（#ff5fa2）
- お気に入り = 背表紙上部のピンクリボン＋ページ右上の**縦長付箋**（後述）

## 1. 絶対原則（かめペース §E — 全画面共通）
- 記録率・網羅率・連続記録・達成演出を出さない
- 未記録・疲労超過に警告色や叱責コピーを使わない（あふれてもそのまま積む）
- **累積疲労の数値・ゲージ・%を出さない**（絵文字の量だけで表現。円グラフ・ランキングの「×N」は項目ごとの個数なのでOK）
- 疲労量の数値入力はさせない

## 2. データマッピング（native/index.html）
内部データは既存の `entries`(from→to) / `tasks` を**無改造**で使う（同期・音・取り込み維持）。
- 日ごとの絵文字列: その日の entries を from 時刻昇順に並べ、各 entry のタスク絵文字を疲労量ぶん繰り返して積む
- 回復系タスク（fh<0）= 回復絵文字（🍵😴🛁🧘 等）。疲労と別系列で保持する
- planned（時刻モードで未来）= グレー絵文字
- 時間帯バケツ: from 時刻で 朝6-9 / 午前9-12 / 午後12-18 / 夜18-（既存ルール）
- お気に入り（fav: dateKey→bool）と日記（diary: dateKey→string）は新規。localStorage 保存（既存の保存機構があればそれに載せる）
- 週番号 = ISO週番号（月曜始まり）。表記は `week29` 形式

## 3. 共通UIパーツ
### 本（背表紙）
- 横66×縦250px（横持ち）／41×168px（縦持ち・グリッド）
- **描画順（奥→手前）: ①絵文字 → ②ガラス枠（白グラデ+insetシャドウのオーバーレイ, z2）→ ③日付・曜日（z3）**。今日/選択枠は z4、付箋リボンは z3
- 地: rgba(255,255,255,.16)、枠線 rgba(27,27,24,.18)、角丸 4px 4px 2px 2px（上だけ丸い）
- 上端に紙の小口ストリップ（高さ9px/5px、横グラデ #f1ede3→#fbf9f3→#efe9dd）
- 棚板: ダークブラウン木目グラデ（#4a3120→#241509、上板は #5a3d27→#33200f）。**縦の木枠は無し・横板は画面端までフル幅**

### 円グラフ（絵文字のかず）
- conic-gradient のドーナツ（74px、中心くり抜き）＋凡例（色チップ・絵文字・名前・×N）
- 項目は個数の多い順。パレット: `#c4f000, #ff5fa2, #e8842c, #7a9a00, #f0c24b, #a08cd0, #6fb8a0, #c9c5b8`（順に割当て）

### 時間帯タイムライン（その日の中身）
- 🌅朝 / ☀️午前 / 🌤午後 / 🌙夜 の見出しごとに行（絵文字＋タスク名＋右端に +N／回復は −N）
- 連続する同一タスクはまとめて +N。グレー（予定）は opacity .45 + grayscale

### 付箋（お気に入りトグル）
- ページ右上、**縦長 20×46px**、rotate(3deg)、ページ上辺から-14pxはみ出す
- ON = ピンク（#ff8fbe→#ff5fa2 グラデ＋上部に半透明の糊帯）、OFF = 点線枠の薄ベージュ（#f3f0e4 / dashed #cfcaba）

### 今日ボタン
- 「日別｜月別」セグメントの隣。押すと: 日別へ切替＋今月＋今日を選択＋今日の本が見える位置へスクロール

## 4. 1a — 縦持ち（本棚ビュー）
- 構成（上から）: ステータスバー / タイトル「がんばりの本棚」+説明 / コントロール行 / 本棚（縦スクロール・初期位置は最下段=今日） / その日バー / タブバー（本棚・シャカ・ホーム・マイ）
- コントロール行: [日別｜月別] セグメント＋今日ボタン＋月セレクト＋ソート（日付順/多い順/少ない順）
- 本棚（日別）: **週=1段・月曜始まり**。月初の段は曜日ぶん左をあけて配置（左詰め、中央寄せしない）。日付順以外のソート時は左詰めで7冊ずつ
- **週番号**: 各段の棚板の左端に `weekNN` を**ゴールド #e8b23a**（Space Mono 8.5px 700）で棚板の上に直接表示
- 本棚（月別）: 表紙（前向きの本 158×260、背は左、角丸 4px 10px 10px 4px、左に折り目ライン）を1段2冊。表紙にその月の絵文字が下からたまる（11px・11列）。月ラベルは上部中央のチップ
- その日バー（常設・タブバーの上）: 左に menu_book＋「その日/その月」のミニタブが上辺に付いた白シート。タイトル＋今日バッジ＋右に「この日を見る/この月を見る」ライムピル
- **その日シート（ポップアップ）**: 本タップ or バータップで**下からスライドイン**（translateY(100%)→0, .3s）。高さ66%、角丸22px上のみ、つまみバー付き、右上に付箋
  - **暗幕・シートともタブバー（64px）は覆わない**。タブバーは常に見え、押せる
  - 中身（日別）: タイトル＋今日バッジ → 時間帯タイムライン → 絵文字のかず（円グラフ） → 日記（textarea, placeholder「この日のことをメモ」） → フッター「この日を見る」
  - 中身（月別）: 月タイトル＋今月バッジ → **疲労ランキング TOP5**（順位・絵文字・名前・×N オレンジ）→ **回復ランキング TOP5**（×N は #7a9a00）→ 円グラフ
  - 暗幕タップで閉じる

## 5. 3a — 横持ち（採用版）＋回転
- 構成: 左に縦ナビ（58px: 本棚=アクティブ、シャカ、ホーム、マイ） / ヘッダ / （日別のみ）フィルタ行 / 棚（横スクロール・初期位置=今日を中央） / 右に本のページ
- ヘッダ: タイトル / [日別｜月別] / 今日 / （日別のみ）表示ピル **[疲労と回復｜寝るまえの量]** / 回転ボタン（screen_rotation）。全要素1行・nowrap
- フィルタ行（日別のみ）: 月セレクト（全部の月は無し・月単位表示） / すべての日・あふれた日（11個以上）・おさまった日 / ソート / 付箋の日
- 棚（日別）: 選択月の1日〜末日を一列に。**月曜の本の下の棚板上に週番号をゴールドで**（週番号は本の下・棚板の上に重ねる、z3）
  - 疲労と回復: 背表紙を左右分割（上部に薄く「疲労/回復」ラベル＋中央に細い区切り線）。左=疲労、右=回復を各列で下から積む（18px）
  - 寝るまえの量: 疲労−回復のぶんを**2列**で下から積む（20px、列間31px）
- 棚（月別）: 表紙 170×280 が並ぶ。フィルタ行は非表示
- 右ページ（その日/その月）: 上下に余白をあけて浮く「本のページ」。紙色 #fffdf8、ページ重なり影（3px/6pxのオフセット紙＋ドロップシャドウ）、左端に menu_book＋「その日/その月」の縦タブ、右上に付箋（日別のみ）
  - 中身は 1a のシートと同じ（日別=タイムライン+円グラフ+日記、月別=TOP5×2+円グラフ）。フッター「この日を見る/この月を見る」
- **回転**: 縦 372×812 ⇄ 横 812×390。縦時: ステータスバー＋ヘッダ＋（日別）ピル/フィルタ行（横スクロール可）＋棚300px＋本のページ＋下タブバー。回転ボタンは両向きに表示

## 6. デザイントークン
- フォント: Zen Kaku Gothic New（本文）/ Space Mono（数値・日付・ラベル）/ Archivo（大数字）
- 色: ライム #c4f000（今日・アクション）、ピンク #ff5fa2（選択・付箋・心）、背景 #f7f4ec、墨 #1b1b18、紙 #fffdf8、サブ文字 #55554e / #9d9b91、罫線 #f1efe8、**ゴールド #e8b23a（週番号）**、オレンジ #e8842c（疲労×N）
- アイコン: Material Symbols Rounded（アクティブは FILL 1）
- 絵文字はシステム絵文字。サイズはモック準拠（棚18-26px、リスト15px、表紙11px）

## 7. 実装しないもの（プロトタイプのみ/本番既存に従う）
- 回転ボタンはモック検証用。本番はデバイスの向き（orientation）で自動切替し、ボタンは出さない
- ログイン/同期/カレンダー取り込み/シャカ音は既存 native の実装をそのまま（本仕様の対象外）
- モックの乱数データ生成コードは移植しない（実データから構成）
