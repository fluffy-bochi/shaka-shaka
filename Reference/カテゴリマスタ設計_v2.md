# カテゴリマスタ設計 v2

v1からの変更：体・心の2軸固有値 ／ コピー式カテゴリ作成 ／ 職業別の大カテゴリ表示 ／ バイト複数選択の平均方式 ／ 軸別モディファイア

---

## 1. 計算モデル

```
疲労量 = { Σ(体固有値 × 補正_体) × 個人係数_体(疲れやすさ)
        ＋ Σ(心固有値 × 補正_心) × 個人係数_心(疲れやすさ) } × 時間
```

- 固有値：小カテゴリごとの（体, 心）/h。表示は合計のみ、内部で2軸保持
- 個人係数：オンボーディングSTEP 5〜8の5段階（0.8 / 0.9 / 1.0 / 1.1 / 1.2 仮）
- 休憩（マイナス方向）は**回復しやすさ係数**を使う（疲れやすさ係数ではない）
- 複数業務選択（バイト等）：`(業務Aの固有値 + 業務Bの固有値 + …) / n` を軸ごとに平均してから補正
  - 割り切り：等分でやったとみなす。少しだけの業務は記録しない自由をユーザーに残す
  - ⚠️ 評価実験で観察：業務追加で合計が下がる挙動に違和感を持つ人がいるか

## 2. カテゴリ作成フロー（ユーザー拡張）

初期値を空欄から自己申告させない。**必ずコピーから始める**（アンカリング）。

```
[+ カテゴリを追加]
 → コピー元を選ぶ（例：講義）
 → 「講義とくらべてどう？」
    かなり楽(×0.6) / 少し楽(×0.8) / 同じ(×1.0) / 少し疲れる(×1.2) / かなり疲れる(×1.5)
    ※体・心それぞれに聞く（2問）。スキップ時は×1.0
 → 名前・絵文字を設定（例：📐 数学）
 → 保存 → コピー元と同じ階層に並ぶ
```

- コピー元のモディファイア構成も引き継ぐ
- 体・心の数値直接編集は「詳細編集」に格納（上級者向け・デフォルト非表示）
- 詳細設定の選択状態＋疲労量はテンプレートとして保存可（例：「月曜1限の数学」）

## 3. 大カテゴリの表示制御

オンボーディングSTEP 4「職業・活動」の回答で初期表示セットを決定。設定画面でトグル可能。

| 職業・活動 | 初期表示（がんばり） |

| 学生 | 授業・学校、課題・勉強、バイト、部活・サークル、いどう、対人 |
| 会社員（デスクワーク） | 仕事、いどう、家事・生活、対人 |
| 立ち仕事・接客 | 仕事、いどう、家事・生活、対人 |
| 医療・介護 | 仕事、いどう、家事・生活、対人 |

| その他 | 全表示 |

非表示カテゴリも検索・類似語チップからは到達可能（隠すだけで消さない）。

## 4. 軸別モディファイア部品

各選択肢が（体係数, 心係数）を持つ。主に効く軸を1つに寄せてUIを単純に保つ。

| 部品 | 選択肢（体, 心） | 主軸 |
|---|---|---|
| M1 対人負荷「だれと」 | ひとり(1.0, 1.0) ／ 気楽な相手(1.0, 1.1) ／ 気をつかう相手(1.0, 1.4) | 心 |
| M2 身体負荷「からだ」 | 座り(0.8, 1.0) ／ 立ち(1.2, 1.0) ／ 動きっぱなし(1.5, 1.0) | 体 |
| M3 時間圧「締切」 | 余裕(1.0, 0.9) ／ ふつう(1.0, 1.0) ／ 追われてた(1.0, 1.3) | 心 |
| M4 忙しさ | ひま(0.9, 1.0) ／ ふつう(1.0, 1.0) ／ ピーク(1.3, 1.2) | 両方 |
| M5 立ち位置 | 聞くだけ(1.0, 0.9) ／ 参加(1.0, 1.0) ／ 自分が回す(1.0, 1.3) | 心 |
| M6 きっかけ（任意・折りたたみ） | 自分で決めた ／ 頼まれた ／ 断れなかった | 係数なし・分析フラグ |

固有モディファイア例：
- 混雑：そこそこ(1.0, 1.0) ／ 混み(1.2, 1.3) ／ 激混み(1.4, 1.6)
- クレーム：なし(1.0, 1.0) ／ あり(1.0, 1.5)
- 当てられ：なし(1.0, 1.0) ／ あり(1.0, 1.3)

## 7. 未決事項

1. 個人係数の5段階の実数値（0.8〜1.2は仮。合計値のレンジ感と合わせて調整）
2. 散歩・軽い運動の軸符号混合（体マイナス・心プラス）を許すか、休憩側は回復のみに単純化するか
3. コピー時の相対調整を体・心の2問にするか、1問（両軸一括）にするか——2問が理屈だが、記録開始前の摩擦を増やす
4. バイトの業務リストをユーザー職場に合わせて編集する導線（初回バイト記録時に「あなたのバイトでやる業務を選んでおく」プリセット化があると2回目以降が速い）
5. M6きっかけフラグの分析設計（「断れなかった」×「疲労持ち越し」クロスをPhase 3のどの指標と対応させるか）
